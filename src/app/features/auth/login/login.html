Termos de Uso e Política de Privacidade (LGPD)

Este é um projeto acadêmico desenvolvido para fins educacionais no âmbito do programa Ford Enter. Nenhum dado real é processado, armazenado em servidores externos ou compartilhado com terceiros.

Em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018), ao utilizar esta plataforma de demonstração, você concorda que:

  • Os dados informados são utilizados apenas para fins de teste e demonstração.
  • Nenhuma informação pessoal é enviada a servidores de produção reais.
  • Você pode solicitar a exclusão dos seus dados a qualquer momento.
  • Este protótipo não realiza cobranças nem coleta dados sensíveis.
